FAQ

· Why another exchange?

BitFrank is 100% on-chain and unstoppable. Everything is processed by the smart contract. There is no server. And you can list any token.

· How can I list a new token?

Just deposit your token (you can find token contract address on Etherscan), and the world can see it in the token list.

· Does that mean I can list scams and spams?

Newly listed tokens are 'unverified' in the token list, and the transaction fee will be higher. You can 'register' the token in the admin panel, to get a lower transaction fee. And admin can 'verify' popular tokens.

· How about transaction fees?

It's 0.2 % for registered / verified tokens. And only the order-taker is charged. The order-maker is not charged. The fee is 0.8 % for unregistered tokens. Note fees are always paid in ETH, so if you are buying token from order, you'll pay a little bit more ETH. Finally because everything is on-chain, there'll be a fixed gas fee for placing / canceling / modifying orders.

· Will this webpage look better in the future?

Definitely. We are working on real-time charts with order books and market depths and event logs, API and market-making bots, and chat.
