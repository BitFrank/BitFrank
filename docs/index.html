FAQ

· Why another exchange?

BitFrank is 100% on-chain and unstoppable. Everything is processed by the smart contract. There is no server. You can even save this webpage to your computer and use it.

· Can I list a new token?

Feel free to list any token. Just deposit your token, and the world can see it in the token list.

· Does that mean I can list scams and spams?

Newly listed tokens are 'unverified' in the token list, and the transaction fee will be higher. You can 'register' the token in the admin panel below, for a lower transaction fee. Or you can wait for admin to 'verify' the token.

· How about transaction fees?

It's 0.1 % for registered / verified tokens. And only the taker is charged. The maker is not charged. The fee is 0.8 % for unregistered tokens. Moreover because everything is on-chain, there'll be gas fee for placing / canceling / modifying orders.

· Will this webpage be more user-friendly in the future?

There will be fancy charts with real-time order books and market depths and event logs. And there will be API and market-making bots. Stay tuned :-)
